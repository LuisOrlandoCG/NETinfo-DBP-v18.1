RECENT BENEFICIARIES

Action

The last 5 beneficiaries that a transfer was made, irrespective of saved or once off, depending on the Customer/CIF selected.

More (link) available.

The following data will be shown:

  • An icon displaying the first letter of the beneficiary name

  • Beneficiary name and account number/alias(if applicable) used for the specific transaction

  • Arrow (link) icon

If no results available display the following in read only format:

'You don’t have any beneficiaries yet'

Actions:

  • MORE: Enabled

  • ADD NEW: Visible and enabled under specific conditions: at least 1 beneficiary is available in the table.

End-user clicks ‘MORE’ button in Recent Beneficiaries.

End-user is navigated to the ‘Beneficiary Maintenance’ page.

End-user clicks ‘ADD NEW’ in Recent Beneficiaries.

End-user is navigated to the ‘Beneficiary Details Input’ page.

RECENT BENEFICIARIES - Select a beneficiary

End-user selects a specific beneficiary.

End-user is redirected to the respective transfer page (input screen) with all the details completed.

When transfer is loaded, the beneficiary appears as New beneficiary.
