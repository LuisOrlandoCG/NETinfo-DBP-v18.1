NETinfo DBP Version 18.1 : Search BIC

Created by Fivi Konstantinou, last modified by 1246-Spyrou Marios on 08, 10, 2024

When the beneficiary account is located at a Bank connected to the SWIFT network, the end-user can select the beneficiary or intermediary bank by searching through a pre-fetched list of all connected banks; upon selection, the following bank details are automatically retrieved;

  • Country

  • City

  • Bank Name

  • BIC/SWIFT Code

  • Branch Name [if available]

  • Branch Address [if available]

  • Branch Code [if available]

Search BIC modal

The figure below (UI/UX - Search BIC), presents the ‘Search BIC’ modal. Referring to this, end-user searches by:

  • Country

  • City

  • Bank Name

  • BIC Code

When end-user selects ‘Search’, the banks matching the search criteria are displayed in table format; user selects the bank of interest and is then redirected to the Fund Transfer Input page (bank details are auto-filled).

End-user can cancel the search and go back using the Back button

Table

Each bank associated with the criteria selected appears as a separate row in the ‘BIC Table’.

The parameters of each bank are shown (in read only format) as columns in the ‘BIC Table’; these are described in the ‘BIC Table’.

End-user can navigate through the pages (forward and backward) if data do not fit in one page.

End-user can define the number of records displayed in a page: Options 10, 25 and 50.

In case of no matching banks, the following message is displayed: ‘No Records Found’.
